#  Banking Dataset Analysis

An end-to-end **Data Analytics** project built using **SQL Server** and **Power BI** to analyze banking transactions, customer accounts, balances, and account performance. The project transforms raw banking data into meaningful insights through SQL queries and interactive dashboards.

---

##  Project Overview

The objective of this project is to analyze a banking dataset to understand customer behavior, transaction patterns, account performance, and overall banking operations. SQL Server was used for data querying and analysis, while Power BI was used to build interactive dashboards for effective business reporting.

##  Key Insights

- Monthly transaction amounts fluctuate throughout the year.
- Debit and Credit transactions are nearly equally distributed.
- Current and Savings accounts show different balance trends.
- Customer demographics provide insights into banking usage.
- Inactive accounts can be identified for customer engagement strategies.
- High-value customers can be recognized based on account balances.
